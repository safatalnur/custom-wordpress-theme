# Alnur Custom Theme

A very minimal lightweight and customizable WordPress theme designed for modern websites using Alnur Theme Basic Skeleton for structure

## Description
This theme provides a responsive built with WordPress best practices. It supports custom templates, theme options.

## Features 
- Customizable header, footer, and sidebar widgets  
- Theme Customizer support  
- Dart Sass included
- French and Bangla language supports for admin panel

## Installation
1. Download or clone this repository:
   ```bash
   git clone https://github.com/username/theme-name.git

2. For Dart Sass compilation, from the root directory, run
   ```bash
   sass --watch sass/style.scss assets/css/alnur-style.css
